The code is a data structure of a class called Student with the attributes name, age, and report card. 
From this, an Arraylist of this same template is created to be able to enter several students through 
a loop, and then show all the students entered with the help of another loop. 

El código es una estructura de datos de una clase llamada Estudiante con los atributos nombre, edad,
y boleta. A partir de esta se crea un Arraylist de esta misma plantilla para poder ingresar varios 
estudiantes por medio de un bucle, para así posteriormente mostrar todos los estudiantes ingresados
con ayuda de otro bucle.
